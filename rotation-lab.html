SENSOR SOURCES
📐 φ (azimuth) — DeviceOrientationEvent.alpha
Compass heading 0–360° → radians
📐 β (pitch) — DeviceOrientationEvent.beta
Front/back tilt –180°–180° → radians
📐 γ (roll) — DeviceOrientationEvent.gamma
Left/right tilt –90°–90° → radians
🌀 ωx — DeviceMotionEvent.rotationRate.beta
Gyroscope x-axis °/s → rad/s
🌀 ωy — DeviceMotionEvent.rotationRate.gamma
Gyroscope y-axis °/s → rad/s
🌀 ωz — DeviceMotionEvent.rotationRate.alpha

⚠️ Requires a physical mobile device and HTTPS.
On iOS: tap START → allow the system permission dialog.
On Android: sensors connect automatically.
HOW TO USE
START → begins recording + live plots.
PAUSE → freezes plots, data is kept.
RESUME → continues adding data.
CLR → wipes all data, resets plots.
Each plot has series toggles and CSV/PNG export.
PINCH on a plot (while paused) to zoom in/out.
DRAG (1 finger, while paused) to pan.
DOUBLE-TAP or tap RESET to return to auto-scale.
Zoom resets automatically when you resume recording.
